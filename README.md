# Bots

This repository contains various discord bots.

This file will contain general information pertaining to the setup while info related to each bot can be found in its corresponding folder.

## Software Setup

Base install of [Anaconda3-2024.10-1-Linux-aarch64](https://repo.anaconda.com/archive/Anaconda3-2024.10-1-Linux-aarch64.sh) or [Anaconda3-2024.10-1-Linux-x86_64](https://repo.anaconda.com/archive/Anaconda3-2024.10-1-Linux-x86_64.sh)

Install the required Python libraries with the included `requirements.txt` file.

``` bash
pip install -r requirements.txt
```
